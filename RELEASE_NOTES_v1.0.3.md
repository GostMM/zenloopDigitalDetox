# Version 1.0.3 Release Notes

## What's New

### Restriction Levels
Choose how strictly you want to block apps during focus sessions:
- Shield Mode: Apps remain visible but blocked with an overlay
- Hide Mode: Apps completely hidden from your home screen
- Automatic difficulty suggestion based on session duration

### Scheduled Sessions Display
View your upcoming scheduled sessions directly on the home screen:
- Time until each session starts
- Start and end times
- Session duration
- List of blocked apps
- Quick cancel option for each session

### Improved Session Planning
- Difficulty selection integrated when scheduling sessions
- Better visual feedback with color-coded indicators
- Compact card design showing all session details at a glance

### Other Improvements
- Refined difficulty selection modal design
- Reduced spacing for better screen space usage
- Complete English and French translations for all new features

# Notes de Version 1.0.3

## Nouveautés

### Niveaux de Restriction
Choisissez l'intensité du blocage pendant vos sessions de concentration :
- Mode Shield : Apps visibles mais bloquées par un overlay
- Mode Hide : Apps complètement masquées de l'écran d'accueil
- Suggestion automatique selon la durée de la session

### Affichage des Sessions Programmées
Visualisez vos sessions programmées directement sur l'écran d'accueil :
- Temps avant le démarrage de chaque session
- Heures de début et de fin
- Durée de la session
- Liste des apps bloquées
- Option d'annulation rapide pour chaque session

### Planification de Sessions Améliorée
- Sélection de difficulté intégrée lors de la planification
- Meilleur retour visuel avec indicateurs colorés
- Design de carte compact affichant tous les détails d'un coup d'oeil

### Autres Améliorations
- Design du modal de sélection de difficulté raffiné
- Espacement réduit pour une meilleure utilisation de l'écran
- Traductions complètes en anglais et français pour toutes les nouvelles fonctionnalités
